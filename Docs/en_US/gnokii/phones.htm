Note:

  • ? - not tested
  • incoming - you have to find another phone supported by mygnokii and send ringtone/logo/etc. to this one
  • no - feature not supported by mygnokii
  • yes - feature supported by mygnokii
  • - - feature not supported by phone

In connection you have types, you should enter in gnokiirc and which will work in phone. When its' in brackets, it means, that this of connection is supported, but "connection" parameter in gnokiirc is not important. ATGSM means, that you should use ATGSM "model" in gnokiirc.
